🚀 Guide d'utilisation

1. Remplir vos plats

Tapez le nom de vos plats dans les cases blanches. L'aperçu à droite se met à jour instantanément.

2. Personnaliser les catégories

Cliquez directement sur les noms grisés (ex: "Pâtes", "Végé") pour les renommer selon vos besoins.

3. Gérer votre semaine

  • 👁️ Icône Œil : Masque ou affiche une journée entière (sans supprimer les données).
  • ✕ Petite croix : Supprime une seule étiquette.
  • + Ajouter un produit : Crée une nouvelle étiquette via des modèles (Pâtes, Poisson...) ou personnalisée.

4. Lancer l'impression

Une fois votre semaine prête, réglez le nombre d'exemplaires et cliquez sur Imprimer.

⚠️ Important : Dans la fenêtre d'impression, réglez l'échelle sur 100% (Taille réelle) et les marges sur Aucune pour que les étiquettes fassent bien 30x50mm.

💡 Astuce : Utilisez le bouton "Afficher les contours" pour imprimer une feuille de test et vérifier l'alignement sur votre papier physique.
